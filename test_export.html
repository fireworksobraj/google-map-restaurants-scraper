Google Maps Scraper - Export Test

This page tests the export functionality.

Instructions:

  1. Load the extension in Chrome:
    • Go to chrome://extensions/
    • Enable "Developer mode"
    • Click "Load unpacked"
    • Select the /workspace/google-maps-scraper folder
  2. Navigate to Google Maps and search for restaurants
  3. Click the extension icon and click "Start Deep Scraping"
  4. Wait for scraping to complete
  5. Click "Download JSON" or "Download CSV" to export data

Fixes Applied:

  • content.js: Fixed saveResults() to properly send all scraped restaurants to background storage
  • background.js: Added SCRAPING_COMPLETE message handler and improved downloadData() with error handling and fallback
  • popup.js: Enhanced error messages for download buttons

Expected Behavior:

  • Restaurants are saved incrementally during scraping AND in final batch
  • Download buttons show specific error messages if no data exists
  • Files download automatically without requiring "Save As" dialog
  • Fallback mechanism works if downloads API fails
